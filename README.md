Calculadora em Python

Este é um projeto simples de calculadora desenvolvido em Python, criado com o objetivo
de aprender lógica de programação, uso de condicionais (`if/elif/else`), loops (`while`)
e tratamento de erros.

O projeto aceita números inteiros e permite realizar várias operações sem
precisar reiniciar o programa.

---

Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Aceita números inteiros
- Tratamento de divisão por zero
- Loop para realizar várias operações seguidas
- Menu numérico (1 = +, 2 = -, 3 = *, 4 = /)

---

Tecnologias utilizadas

- Python 3

---

Como executar o projeto

1. Certifique-se de ter o Python instalado no seu computador  
2. Clone este repositório ou baixe os arquivos  
3. Execute o arquivo.
